### LGMD, EMD, and a Hybrid Spiking Neural Network 

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://python.org)

This repository contains the full implementation for the paper:

> **Comparative Evaluation of Biologically Inspired Motion Detection Models:
> LGMD, EMD, and a Hybrid Spiking Neural Network**
> Vanessa Ndiangang and Pengcheng Liu — University of York, 2025

---

## Overview

Three biologically inspired visual motion detection models are implemented,
evaluated, and compared on programmatically generated synthetic stimuli and
real-world videos from the LGMD2 open-source package:

| Model | Type | Language |
|---|---|---|
| LGMD2 | Lobula Giant Movement Detector | C# / .NET 8.0 |
| EMD | Elementary Motion Detector (Reichardt correlator) | Python |
| Hybrid SNN | Spiking Neural Network (LGMD + EMD inspired) | Python / PyTorch |

## Reproducing the results

### Step 1 — Generate synthetic stimuli
```bash
python scripts/stimulus_generator.py \
    --output_dir videos/synthetic \
    --label_csv  labels/ground_truth.csv
```
Creates 25 videos (5 stimulus types × 5 variants) and the ground truth CSV.

### Step 2 — Add real videos

Copy the 10 LGMD-dataset videos into `videos/real/`. These are available
from the LGMD2 open-source package (Fu et al., 2020):

```
videos/real/
    loom_on_inhibition.mp4
    looming_grouping.mp4
    looming_off_inhibition.mp4
    looming_photoreceptor.mp4
    raw_loom.mp4
    recession_grouping.mp4
    recession_off_inhibition.mp4
    recession_on_inhibition.mp4
    recession_photoreceptor.mp4
    recession_raw.mp4
```

### Step 3 — Run all models and evaluate
```bash
python compare_models.py
```

This trains the SNN, runs all three models on synthetic and real videos,
and produces `results/evaluation_summary.csv`.

To skip training if weights already exist:
```bash
python compare_models.py   # auto-detects existing weights
```

To skip model runs and go straight to evaluation:
```bash
python compare_models.py --eval_only
```

### Step 4 — Generate figures
```bash
python generate_figures.py
python generate_stimulus_figure.py
```

All figures are saved to `figures/` as PDF (for Overleaf) and PNG.

## Key results

| Model | Accuracy | Precision | Recall | F1 |
|---|---|---|---|---|
| LGMD2 | 0.55 | 0.09 | 0.001 | 0.002 |
| EMD | 0.51 | 0.29 | 0.07 | 0.11 |
| Hybrid SNN | 0.51 | 0.49 | 0.96 | 0.65 |

The hybrid SNN achieves **precision = 1.00 on all looming stimulus types**
(zero false positives). The LGMD fires at frames 9–11 before the
magnitude-based ground truth threshold, reflecting biological rate sensitivity.
The EMD produces near-zero motion energy on looming stimuli, confirming
directional cancellation as predicted by the Reichardt mechanism.
